# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

* `GET /api/users`
* `GET /api/users/:id`
* `POST /api/users`
* `PATCH /api/users/:id`

### Session

* `POST /api/session`
* `DELETE /api/session`
* `GET /api/session`

### Routes

* `GET /api/routes/:id`
* `GET /api/users/:id/routes`
* `POST /api/routes`
* `PATCH /api/routes/:id`
* `DELETE /api/routes/:id`

### Workouts

* `GET /api/routes/:id`
* `GET /api/users/:id/routes`
* `POST /api/routes`
* `PATCH /api/routes/:id`
* `DELETE /api/routes/:id`
